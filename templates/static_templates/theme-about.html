## About Section ##

About PolimiX

PolimiX is developing new online course material in order to improve on-campus courses and provide broader public access to exciting learning opportunities.

The platform is running and developing on an instance of the OpenEdX platform.

## FAQ Section ##

FAQs

Has Politecnico di Milano joined the edX consortium?
No. We're using the open source software OpenEdx.

What browsers are recommended for use with PolimiX?
The latest versions of Google Chrome, Mozilla Firefox, and Apple Safari. (The site is currently not optimized for mobile use.)
